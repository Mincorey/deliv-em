# REQUIREMENTS: Deliv'em

## Table Stakes (must-have for launch)
- [ ] Аутентификация через Supabase Auth (email + password)
- [ ] Регистрация с выбором роли (customer/courier)
- [ ] Защищённые маршруты (redirect to /auth если не залогинен)
- [ ] Профиль пользователя (просмотр + редактирование)
- [ ] Дашборд с адаптацией по роли
- [ ] CRUD поручений (создание, просмотр, статусы)
- [ ] Лента заданий для курьера (фильтрация по типу/городу)
- [ ] Принятие и выполнение поручения курьером
- [ ] Реалтайм чат внутри поручения
- [ ] Кошелёк с историей транзакций
- [ ] Рейтинговая система (1-5 звёзд + комментарий)
- [ ] Уведомления в реальном времени
- [ ] Избранные (курьеры/заказчики)
- [ ] Архив завершённых поручений
- [ ] Полная мобильная адаптация
- [ ] Feedback форма (сохраняется в Supabase)

## Core Features

### Auth & Profiles
- Email/password auth через Supabase
- При регистрации: роль, имя, фамилия, телефон, email, город
- Для курьера: тип транспорта
- Автосоздание записи в `profiles` через Supabase trigger
- Редактирование профиля: фото (Supabase Storage), данные, транспорт, bio
- Смена пароля

### Tasks (Поручения)
- Создание: тип, название, описание, откуда/куда, вознаграждение, срок
- Публикация: списание 100₽ с кошелька (placement_fee)
- Назначение конкретному курьеру (task_invitations)
- Статусы: draft → published → matched → in_progress → completed/cancelled
- Детальная страница с картой (Яндекс)
- Курьер меняет статус: принять → начать → завершить
- Заказчик может отменить (если не принято)

### Wallet (Кошелёк)
- Текущий баланс
- Пополнение через AnyPay (200/500/1000₽ + произвольная сумма)
- История транзакций с пагинацией
- Автосписание 100₽ при публикации поручения
- Выплата курьеру при завершении (reward)

### Chat (Сообщения)
- Список чатов (по поручениям)
- Realtime сообщения через Supabase Realtime
- Счётчик непрочитанных
- is_read / mark as read

### Ratings
- После завершения — оба участника оценивают друг друга
- 1-5 звёзд + опциональный комментарий
- Автопересчёт рейтинга курьера (trigger в БД)

### Couriers Directory
- Список всех курьеров сервиса
- Сортировка по рейтингу
- Фильтр по городу и транспорту
- Карточка курьера: фото, рейтинг, кол-во выполненных, bio

### Notifications
- Realtime уведомления (badge в топбаре)
- Типы: task_published, task_accepted, task_completed, message, rating
- Список уведомлений с пометкой прочитанных

## Technical Requirements
- Next.js 15 App Router + TypeScript
- Tailwind CSS (полный config из прототипа, не CDN)
- Supabase JS SDK v2
- Server Components для начальных данных
- Client Components для интерактивности и realtime
- Server Actions для мутаций
- Middleware для auth redirect
- Zod для валидации форм
- React Hook Form
- Supabase Storage для аватаров
- Environment variables: NEXT_PUBLIC_SUPABASE_URL, NEXT_PUBLIC_SUPABASE_ANON_KEY

## Out of Scope (v1)
- Мобильное приложение (отдельный проект)
- SMS уведомления
- Геолокация курьера в реальном времени на карте
- Верификация документов курьера
- Промокоды
- Поддержка нескольких валют
